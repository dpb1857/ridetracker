
Type 'make python' to build the local virtualenv python to run this project.

If you need to build python for your system, you can go into the build-python
directory and type 'make python' as root to configure and install python in
/usr/local/python-<version>.

To run the pbp2011 results app:

cd .../ridetracker
. python/bin/activate
cd tracker
./manage.py runserver
